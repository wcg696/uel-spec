# uel-spec
Universal Execution Language (UEL) - Draft Specification
## Reference Runtime

- **AgentTor Core (ROS2 + MoveIt Runtime)**  
  https://github.com/wcg696/AgentTor-Core

This is the official reference implementation of UEL.

It demonstrates:
- Natural language → UEL compilation
- Constraint-aware execution
- ROS2 + MoveIt integration
- Simulation with UR5e + gripper

Developers can use it as a starting point to build their own adapters and runtimes.
